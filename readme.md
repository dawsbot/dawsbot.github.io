## FAQ

#### How'd you make it look like a readme file?

I used Sindresorhus' [github-markdown-css](https://github.com/sindresorhus/github-markdown-css) to create an html doc that appears like a GitHub readme file. The way that information appears and is organized in a readme is efficient. I wanted that efficiency in my personal site, and besides, this gives extra geek points.
